# Affidea Setúbal Website

**Client:** Affidea Setúbal — Clínica Privada Multidisciplinar
**Location:** Av. Antero de Quental 2, Setúbal, Portugal
**Phone:** 265 528 060
**Rating:** 4.6★ (1845 reviews)

## About

Modern, responsive website for a private clinical facility offering diagnostic services, laboratory analysis, imaging, and medical consultations. Designed with professional, trustworthy aesthetic suitable for healthcare sector.

## Features

- Fully responsive design (mobile, tablet, desktop)
- Modern gradient hero section with clear CTAs
- Service cards with icons
- About section with company mission
- "Why Us" section highlighting key differentiators
- Integrated contact form
- Business hours display
- Accessibility-focused design
- Smooth scroll animations (reveal on scroll)

## Styling

- **Primary Color:** #0066CC (Professional blue)
- **Accent Color:** #FF6B35 (Warm orange, for highlights)
- **Typography:** Montserrat (titles) + Open Sans (body)
- **Design Pattern:** Card-based, clean, professional, acolhedor

## Form

Contact form submits to PageFactory email service. Includes:
- Name, Email, Phone
- Service selection dropdown
- Message textarea
- Consent checkbox with privacy policy link

## Deployment

Ready for production deployment to Vercel with nodemailer integration for form submissions if needed.
